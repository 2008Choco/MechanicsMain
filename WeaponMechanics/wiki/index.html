About the plugin

WeaponMechanics is a free guns plugin that allows the creation and complete customization of guns through the use of custom projectiles, custom explosion algorithms, well thought out weapon handlers, custom trigger systems, and so much more. Key features of WeaponMechanics that you won't see anywhere else:

  1. Thermal Scopes
  2. Projectiles Passing Through Things
  3. Shaped Explosions
  4. Shaped Spread from Sprites

Getting started

After downloading the plugin and running it on your server, you should take a look at the wiki. The wiki will help you learn how to get all of the features and apply them to guns.

In the wiki you will see words in diamonds, which look like: <>. Here's what they mean:

  • <Integer>: A number without a decimal
    • 7
    • -42
    • 709
  • <Double>: A number with a decimal
    • 0.0
    • -304.6
    • 1.789630366652
  • <String>: A set of characters surrounded by quotes (""). This can contain numbers.
    • "My name is bob"
    • "ZOMBIE"
    • "CREEPER 32 Hi 7"
  • <Boolean>: A true or false statement. Use lowercase letters
    • true
    • false

Sometimes, you will see multiple words in a diamond, like <String~Double~Double>. In this situation, you need to put everything in quotes, and then follow the rules above. The tilde ~ character can be replaced with either the tilde (~), the dash (-), or the space ( ). Valid examples of this:

  • "ENTITY_GENERIC_HURT~3.0~0.52"
  • "Whatever 42.9 0.5"
  • "Word--32-43"

Questions and Answers

Question: Is this a crackshot plugin addon?
Answer: No, this is a standalone plugin. It does not require crackshot to run.

Question: Is this plugin premium?
Answer: No, this is a free plugin. Premium addons may be developed in the future, but the base plugin will remain free forever.
